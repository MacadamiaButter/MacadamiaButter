# Hi, I'm Butter 🧈

I build **privacy-first, local-first software** — tools that work with the
internet unplugged. Everything I run is self-hosted: local LLMs on my own GPU,
encrypted storage, no cloud accounts, no telemetry.

I write about it at **[localfirstlab.org](https://localfirstlab.org)** — hands-on
comparisons of private, self-hosted alternatives to everyday SaaS, from daily use
of the actual tools.

### Projects

- **[Journal-Lens](https://github.com/MacadamiaButter/Journal-Lens)** — analyze your
  private journal with a *local* LLM. FastAPI + any OpenAI-compatible local server
  (LM Studio, Ollama). Your words never leave your machine.
- **[python-toolbox](https://github.com/MacadamiaButter/python-toolbox)** — small,
  dependency-light Python utilities for file wrangling, log monitoring, and data
  conversion. The kind of scripts I build for clients.
- **[local-claude-proxy](https://github.com/MacadamiaButter/local-claude-proxy)** —
  point Anthropic-style tooling at a fully local model.

### Work with me
